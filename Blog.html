Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a webpage as a tree of objects (nodes).
It allows developers to manipulate HTML elements, attributes, and content dynamically using JavaScript.
Through the DOM, you can modify the content, structure, and styling of a webpage in real-time.

How do you select an element from the DOM?

To select an element from the DOM:
1. Use `document.getElementById("id")` to get an element by its ID.
2. Use `document.querySelector("selector")` to select the first element matching a CSS selector.
3. Use `document.getElementsByClassName("class")` to select elements by class name.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events triggered by its child elements. Instead of adding event listeners to each child, the event bubbles up to the parent, which can then determine which child triggered the event.
It is useful because:
1. It improves performance by reducing the number of event listeners.
2. It simplifies code, especially for dynamically added elements.
3. It ensures that newly created elements are automatically handled without needing additional listeners.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate attributes, use setAttribute() to change an attribute or getAttribute() to retrieve one. For styles, directly modify the style property, like element.style.color = "blue".
